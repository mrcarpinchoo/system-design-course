Components of the InternetComponentes de Internet

The Internet is a network of networks, organized in three main components.Internet es una red de redes, organizada en tres componentes principales.

Network EdgeBorde de Red

End systems (hosts): clients and servers that run applications.Sistemas finales (hosts): clientes y servidores que ejecutan aplicaciones.

Includes laptops, smartphones, IoT devices, web servers, and data centers. These are the source and destination of all network communication.Incluye laptops, smartphones, dispositivos IoT, servidores web y centros de datos. Son el origen y destino de toda la comunicación de red.

Access NetworksRedes de Acceso

Connect end systems to the first router (edge router).Conectan los sistemas finales al primer router (router de borde).

Technologies include DSL, cable, fiber (FTTH), WiFi, and cellular networks. Bandwidth and latency vary widely between access network types.Las tecnologías incluyen DSL, cable, fibra (FTTH), WiFi y redes celulares. El ancho de banda y la latencia varían ampliamente entre tipos de redes de acceso.

Network CoreNúcleo de Red

Interconnected routers forming the backbone of the Internet.Routers interconectados que forman la columna vertebral de Internet.

Tier-1 ISPs (backbone providers) interconnect through Internet Exchange Points (IXPs). Data travels across multiple autonomous systems via BGP routing.Los ISP de nivel 1 (proveedores de backbone) se interconectan a través de puntos de intercambio de Internet (IXPs). Los datos viajan a través de múltiples sistemas autónomos mediante enrutamiento BGP.

Access NetworksRedes de Acceso

Home NetworksRedes Domésticas

WiFi router connects laptops, phones, smart TVs, and IoT devices to the ISP.El router WiFi conecta laptops, teléfonos, smart TVs y dispositivos IoT al ISP.

Typical setup: modem → router → WiFi AP. Speeds range from 25 Mbps (DSL) to 1+ Gbps (fiber). WiFi 6/6E supports higher density and lower latency.Configuración típica: módem → router → AP WiFi. Velocidades de 25 Mbps (DSL) a 1+ Gbps (fibra). WiFi 6/6E soporta mayor densidad y menor latencia.

Institutional NetworksRedes Institucionales

Ethernet switches, routers, firewalls, and servers connected via dedicated links.Switches Ethernet, routers, firewalls y servidores conectados mediante enlaces dedicados.

Enterprise networks use VLANs for segmentation, redundant links for high availability, and 10/25/100 GbE for backbone connectivity.Las redes empresariales usan VLANs para segmentación, enlaces redundantes para alta disponibilidad y 10/25/100 GbE para conectividad de backbone.

Mobile NetworksRedes Móviles

Cell towers provide wireless coverage via 4G LTE and 5G standards.Las torres celulares proveen cobertura inalámbrica mediante estándares 4G LTE y 5G.

5G offers up to 10 Gbps peak speeds, <1ms latency, and massive device density. Uses mmWave for short range and sub-6 GHz for broader coverage.5G ofrece hasta 10 Gbps de velocidad pico, <1ms de latencia y densidad masiva de dispositivos. Usa mmWave para corto alcance y sub-6 GHz para cobertura amplia.

Network CoreNúcleo de Red

Packet SwitchingConmutación de Paquetes

Data split into packets; each routed independently. Used by the Internet.Datos divididos en paquetes; cada uno enrutado independientemente. Usado por Internet.

More efficient for bursty traffic. Statistical multiplexing allows many users to share links. Packets may arrive out of order and need reassembly.Más eficiente para tráfico en ráfagas. La multiplexación estadística permite a muchos usuarios compartir enlaces. Los paquetes pueden llegar desordenados y necesitan reensamblaje.

Circuit SwitchingConmutación de Circuitos

Dedicated path reserved for the entire communication. Traditional telephony.Ruta dedicada reservada para toda la comunicación. Telefonía tradicional.

Guaranteed bandwidth and constant delay, but wasteful when data is not being transmitted. Resources reserved even during silence periods.Ancho de banda garantizado y retardo constante, pero desperdicia recursos cuando no se transmiten datos. Recursos reservados incluso durante periodos de silencio.

HTTP / HTTPSHTTP / HTTPS

HTTPS = HTTP + TLSHTTPS = HTTP + TLS

HTTPS encrypts all communication using TLS (Transport Layer Security), protecting data integrity and confidentiality.HTTPS cifra toda la comunicación usando TLS (Seguridad de Capa de Transporte), protegiendo la integridad y confidencialidad de los datos.

TLS handshake: Client Hello → Server Hello + Certificate → Key Exchange → Encrypted session. Uses asymmetric crypto for key exchange, symmetric crypto for data transfer. Port 443 (vs 80 for HTTP).Handshake TLS: Client Hello → Server Hello + Certificado → Intercambio de claves → Sesión cifrada. Usa criptografía asimétrica para intercambio de claves, simétrica para transferencia de datos. Puerto 443 (vs 80 para HTTP).

SummaryResumen

Internet ArchitectureArquitectura de Internet

The Internet is structured in three layers: network edge (end systems), access networks (connecting users), and network core (routing backbone). Packet switching enables efficient resource sharing.Internet se estructura en tres capas: borde de red (sistemas finales), redes de acceso (conectando usuarios) y núcleo de red (backbone de enrutamiento). La conmutación de paquetes permite compartir recursos eficientemente.

Protocol LayersCapas de Protocolos

The OSI model defines 7 layers for network communication. TCP/IP simplifies this to 4 practical layers. Each layer encapsulates data and provides services to the layer above.El modelo OSI define 7 capas para comunicación de red. TCP/IP simplifica esto a 4 capas prácticas. Cada capa encapsula datos y provee servicios a la capa superior.

TCP vs UDPTCP vs UDP

TCP provides reliable, ordered delivery for web and email. UDP offers fast, lightweight communication for streaming and gaming. HTTPS adds TLS encryption on top of HTTP for secure web traffic.TCP provee entrega confiable y ordenada para web y correo. UDP ofrece comunicación rápida y ligera para streaming y juegos. HTTPS agrega cifrado TLS sobre HTTP para tráfico web seguro.
